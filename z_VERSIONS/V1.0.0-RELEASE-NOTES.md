# Version 1.0.0 Release

## Major Features

### Merge Cluster Feature
- **Manual Cluster Merging**: Click any parent image in Visual Analysis to open merge modal
- **Visual Selection**: Scroll through all other parent images and select target with orange border highlight
- **Complete Merge**: Merges source cluster (including all brackets, derivatives, and similar reps) into target cluster
- **Automatic Updates**: Refreshes Visual Analysis table after merge

### Bracket & Derivative Handling
- **Automatic Bracket Detection**: When extracting images from super clusters, all related bracket images are automatically moved together
- **Derivative Preservation**: All derivatives associated with images are preserved during extraction and merging
- **Smart Grouping**: Related similar reps that share bracket images are automatically grouped

### CLIP Setup Progress Modal
- **Real-time Progress**: Beautiful progress modal shows setup progress during first-time CLIP service installation
- **Time Estimates**: Displays elapsed time and estimated remaining time
- **Step Indicators**: Visual indicators for each setup phase (venv, dependencies, model download, service start)
- **Auto-hide**: Automatically hides when setup completes

### Package Installation Improvements
- **Automatic Verification**: Verifies critical packages (like torchvision) after installation
- **Auto-upgrade**: Automatically upgrades existing venvs if packages are missing
- **Better Error Handling**: More informative error messages with detailed logging

## Improvements

- Fixed prompt strategy selection to respect dropdown selection (no more overrides)
- Improved CLIP service startup in packaged apps with better timeout handling
- Enhanced GPS extraction for super clusters (shows GPS if any image in cluster has it)
- Better error handling and logging throughout the application
- Improved CLIP service health checks with timeout prevention

## Technical Updates

- Upgraded CLIP model to `clip-vit-huge-patch14` with explicit GPU configuration (Apple Silicon MPS / NVIDIA CUDA)
- Improved CLIP service startup process (non-blocking, asynchronous)
- Enhanced Python virtual environment handling in packaged apps
- Better model caching detection for Hugging Face models
- Improved bracket relationship detection and management

## Installation

1. Download the DMG from the releases page
2. Mount and drag the app to Applications
3. On first run, the CLIP setup progress modal will appear (this is a one-time setup)
4. Wait for setup to complete (may take 5-10 minutes on first run)
5. The app is ready to use!
